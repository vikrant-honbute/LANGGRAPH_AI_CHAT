# LANGGRAPH_AI_CHAT

A Python-based AI chat project built with LangGraph.

## Overview

`LANGGRAPH_AI_CHAT` focuses on building conversational AI workflows using graph-based orchestration.  
It is implemented fully in Python.

## Tech Stack

- **Language:** Python (100%)
- **Framework:** LangGraph
- **Type:** AI Chat Application

## Prerequisites

- Python 3.10+ (recommended)
- `pip` package manager

## Installation

```bash
git clone https://github.com/vikrant-honbute/LANGGRAPH_AI_CHAT.git
cd LANGGRAPH_AI_CHAT
```

Install dependencies (use whichever file exists in your repo):

```bash
pip install -r requirements.txt
```

## Configuration

Create a `.env` file (if your code uses environment variables), for example:

```env
GROQ_API_KEY=your_api_key_here
```

## Run

Use your project’s actual entry-point file:

```bash
python <your_entry_file>.py
```

Example (only if applicable in your repo):

```bash
python app.py
# or
python main.py
```

## What this project does

- Creates a chat workflow using LangGraph
- Handles conversational state across interactions
- Generates AI responses through configured LLM provider(s)
